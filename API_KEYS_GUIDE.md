Go to **\"Database\"** (Clusters)\n2. Click **\"Connect\"** on your cluster\n3. Choose **\"Connect your application\"**\n4. Driver: **Node.js**, Version: **4.1 or later**\n5. Copy the connection string:\n```\nmongodb+srv://brandtracker_user:<password>@brand-tracker-cluster.xxxxx.mongodb.net/?retryWrites=true&w=majority\n```\n6. Replace `<password>` with your actual password\n7. Add database name: `/brandtracker` before the `?`\n\n**Final MongoDB URI Example:**\n```bash\nMONGODB_URI=mongodb+srv://brandtracker_user:AutoGen_P@ssw0rd123@brand-tracker-cluster.abc123.mongodb.net/brandtracker?retryWrites=true&w=majority\n```\n\n---\n\n## üîë API Keys Setup Guide\n\n### 1. ü§ñ OpenAI API Key (REQUIRED)\n\n**What it's for:** AI-powered sentiment analysis\n\n**Steps:**\n1.
